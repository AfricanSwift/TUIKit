RenderStyle

public enum RenderStyle

Unicode Render Style - drawille: Braille character rendering - short: Short ASCII ramp (paulbourke.net) using color - short2: Short ASCII ramp (iJoshSmith) using color - long: Long ASCII ramp (www.ludd.luth.se) using color - ditherShort: Short ASCII ramp using braille value - ditherLong: Long ASCII ramp using braille value - block: Block ramp using braille value - block2: Block ramp using color & braille value
